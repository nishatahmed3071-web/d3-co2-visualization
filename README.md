# D3 CO₂ Emissions Visualization - Pie Chart

##  Project Overview
Interactive pie chart visualizing CO₂ emissions distribution among the top 5 emitting countries (2020 data).

##  Links
- **GitHub Repository:**   https://github.com/nishatahmed3071-web/d3-co2-visualization.git
- **Live Deployment:** https://nishatahmed3071-web.github.io/d3-co2-visualization/

##  Visualization Type: Pie Chart
This project implements an **interactive pie chart** showing:
- Top 5 CO₂ emitting countries (2020)
- Percentage distribution of emissions
- Total emissions displayed
- Interactive hover effects and tooltips
